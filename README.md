# Mold demo website — famous papers as highlighted pages

Static site: landmark ML papers (2013-2025) + FARS AI positive controls, each rendered
as paper pages with the molds' evidence highlighted in place (Korean UI + English toggle,
rich hover tooltips, light/dark theme).

- `index.html` — year-vs-mold-index timeline + table (entry point)
- `pages/*.html` — 15 per-paper highlighted pages ("every mold of one paper")
- `molds/index.html` — **mold-type viewer** ("every paper of one mold"): per-mold pages
  with 15-paper verdicts and real source-text example browsers, and **interactive graphs**
  for the cross-sentence/section molds:
  - `molds/CRN.html` — section-reference network (nodes=sections, edges=cross-section \ref)
  - `molds/MR.html` — macro-redundancy arcs (edges=claim pairs restated across sections)
  - `molds/CP.html` — work-work citation graph (edges=same-sentence co-citations)
  - `molds/ARG.html` — argument flow (Qwen2.5-32B typed nodes/edges, addresses-loops)
- `combined_demo.html` — single-file version (tabs; used for the claude.ai artifact)

All pages come in Korean (`X.html`) and English (`X_en.html`) with an in-page toggle.

Generated by `Evaluation/famous_paper_demo/{run_demo.py, make_pages.py, make_mold_viewer.py,
build_combined.py}` in the artifact-ai2science repo.
Measurements audited in `Evaluation/famous_paper_demo/verify/AUDIT.md` (2026-07-14, v2).
Every highlighted example is a real slice of the paper's LaTeX source (span.text ==
source[start:end]) — nothing fabricated.
